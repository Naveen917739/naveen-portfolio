Web Application · Python · Flask

ReqTrack – Requirements Management System

ReqTrack is a full-stack web application designed for software development teams to manage and track project requirements through their lifecycle. The system allows team leads to create requirements, assign them to developers, set priorities, and track completion status in real time.

Key Features

  • User authentication with role-based access (Admin, Lead, Developer)
  • Requirements CRUD with status tracking (Open → In Progress → Completed)
  • Priority levels and due date management
  • Project dashboard with progress charts
  • Comment threads on each requirement
  • Export to Excel/PDF reports
